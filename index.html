Atenção

O projeto Food Delivery está sendo mostrado como um site perigoso em alguns navegadores.

Este aviso está aparecendo devido a uma funcionalidade que realiza o checkout de pagamento para finalizar as compras no site. Porém, o checkout não é válido, e nenhuma compra é realizada de fato.

Por ser um projeto pessoal, a funcionalidade é só para exemplificar o funcionamento completo de um e-commerce. Então caso apareça pra você, não se preocupe com o aviso e pode prosseguir que o site é seguro e não recolhe nenhuma informação pessoal.

Para prosseguir basta clicar em "DETALHES" e escolher a opção para continuar.
